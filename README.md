# Online Course Marketplace

A static web project showcasing an online course marketplace experience with course catalogs, instructor profiles, cart interactions, theme toggle, user authentication UI, and a polished dashboard-style interface.

## Project Overview

This project includes:
- `index.html` — page structure and navigation
- `style.css` — responsive styling, glassmorphism cards, dark mode support, and layout design
- `script.js` — client-side data seeding, state management, routing, UI rendering, and interaction logic
- `assets/` — supporting static assets such as the logo

## Features

- Course browsing by category and instructor
- Simulated user authentication and dashboard navigation
- Shopping cart indicator and quick UI feedback
- Dark/light theme toggle
- Responsive header, navigation, and mobile-friendly layout
- Toast notifications and a loader animation

## Run Locally

1. Open `index.html` directly in a browser.
2. Or use a local server for best results:
   - With VS Code Live Server, open the project and start the server.
   - Or use a simple static server tool.

## Notes

- This is a frontend-only static site.
- Data is seeded in `script.js` and stored in `localStorage`.
- No backend or build step is required.
